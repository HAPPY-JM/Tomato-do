profile.js -> 프로필 화면에서 작동할 js

modules/bio.js -> 이름과 자기소개 보여줄 html

	      (이름, 자기소개 적은거 데이터베이스에 저장, 수정하기)
EditBio.js -> 이름과 자기소개 적는 칸 보여줄 html 
nav.js -> 위 메뉴
report.js -> 지금은 토마토 사진인데 리포트내용을 보여줄 html
